# Change Log

## 0.2.0

- Initial release of the DevFlow VS Code / Cursor extension.
- Commands: Start Workflow, Initialize, Doctor, Resume Last Run, Refresh Runs,
  Open Run Report.
- Activity Bar "Runs" tree view showing each run's 12 stages with live status
  from `.devflow/runs/*/state.json`.
- Status bar item summarizing the latest run.
- Bundles the `devflow` CLI inside the extension (no separate install required).
